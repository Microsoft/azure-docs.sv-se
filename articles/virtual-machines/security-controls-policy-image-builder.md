# <a name="azure-policy-regulatory-compliance-controls-for-azure-image-builder"></a>Azure Policy kontroll av efterlevnad för Azure Image Builder

Regelefterlevnad [i Azure policy](../governance/policy/concepts/regulatory-compliance.md) ger Microsoft skapade och hanterade initiativ definitioner, som kallas _inbyggda moduler_, för **domäner** och **säkerhets kontroller** som är relaterade till olika efterlevnads standarder. Den här sidan listar **domänerna för efterlevnad** och **säkerhets kontroller** för Azure Image Builder. Du kan tilldela de inbyggda programmen för en **säkerhets kontroll** individuellt för att hjälpa dina Azure-resurser att vara kompatibla med den aktuella standarden.
