# <a name="azure-policy-regulatory-compliance-controls-for-azure-machine-learning"></a>Azure Policy kontroll av efterlevnad för Azure Machine Learning

Regelefterlevnad [i Azure policy](../governance/policy/concepts/regulatory-compliance.md) ger Microsoft skapade och hanterade initiativ definitioner, som kallas _inbyggda moduler_, för **domäner** och **säkerhets kontroller** som är relaterade till olika efterlevnads standarder. Den här sidan listar **domänerna för efterlevnad** och **säkerhets kontroller** för Azure Machine Learning. Du kan tilldela de inbyggda programmen för en **säkerhets kontroll** individuellt för att hjälpa dina Azure-resurser att vara kompatibla med den aktuella standarden.
